A template for building a long-lasting online catalog of creative work —
design, engineering, art, writing, anything. Inspired by the *catalogue
raisonné*, the authoritative listing of an artist's complete output.

Fork the repo, add your projects as Markdown files, connect a free
Cloudflare or GitHub Pages account, and get a published catalog with an
index and a detail page for every project. No servers, no databases,
no build tools required locally.

Media lives on storage you control — either alongside the repo (GitHub Pages)
or in Cloudflare R2. The site itself is a handful of plain HTML files,
readable in a browser in thirty years with no toolchain at all.
