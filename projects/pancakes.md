Documentation scatters. Wikis nest, note apps branch, and the thing you wanted someone to read ends up four clicks deep. Pancakes refuses the tree. It takes a pile of documents and stacks them into a single scrollable page — flat, readable, done. The name is literal: each doc lands on top of the last.

The stack is made for handing over, not hoarding. A whole project's documentation becomes one link, and it can live inside an organization's own walls when access needs limiting. Built on Google Apps Script and the Google Docs API.
